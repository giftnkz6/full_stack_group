Router Description

Method Path Description
get / To render the home page (index.html)
get /users To retrieve all users
get /user/:id To retrieve a single user by using id
post /register To create a new user
post /login Allows a user to login
put /user/:id To update the user by id
delete /user/:id To delete a user by id
get /products To retrieve all products
get /product/:id To retrieve a single product by id
post /product To create a new product
put /product/:id To update a product
delete /product/:id To delete a product
